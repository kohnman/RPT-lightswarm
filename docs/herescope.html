Herescope Integration Guide

Complete guide for integrating with Unreal Engine application

‚ö° Quick Start (Test in 60 Seconds)

Run these 3 commands in Terminal to verify the API is working:

# 1. Check system is running
curl http://MIDDLEWARE_IP:3000/api/v1/health
# 2. Login (stops ambient animation)
curl -X POST http://MIDDLEWARE_IP:3000/api/v1/session/login \
-H "Content-Type: application/json" \
-d '{"agentId": "test-agent"}'
# 3. Light an apartment green (Available)
curl -X PUT http://MIDDLEWARE_IP:3000/api/v1/apartments/RPT-L09-01 \
-H "Content-Type: application/json" \
-d '{"state": "AVAILABLE"}'

Replace MIDDLEWARE_IP with: localhost (same machine), 192.168.x.x (local network), or your Railway URL (remote)

API Endpoints

POST /api/v1/session/login Agent login - stops ambient

Request Body:

{
"agentId": "agent-001"
}

Response:

{
"success": true,
"message": "Session started"
}
What happens: Ambient animation stops ‚Üí All lights fade down from top floor to bottom
POST /api/v1/session/logout Agent logout - starts ambient
What happens: All apartment lights turn off ‚Üí Ambient animation starts looping
PUT /api/v1/apartments/{apartmentId} Light single apartment

Request Body:

{
"state": "AVAILABLE",
"fadeTime": 500,
"intensity": 200
}
Parameter Type Required Description
state string Yes AVAILABLE, SOLD, RESERVED, UNAVAILABLE, SELECTED
fadeTime number No Fade duration in milliseconds (default: 500)
intensity number No Brightness 0-255 (default: 255)
Apartment ID Format: RPT-L{floor}-{unit} (e.g., RPT-L09-01)
PUT /api/v1/apartments/batch Light multiple apartments

Same state for all:

{
"apartmentIds": ["RPT-L09-01", "RPT-L09-02"],
"state": "AVAILABLE"
}

Different states:

{
"apartments": [
{"id": "RPT-L09-01", "state": "AVAILABLE"},
{"id": "RPT-L09-02", "state": "SOLD"}
]
}
PUT /api/v1/floorplates/{floorplateId} Light entire floor

Request Body:

{
"state": "AVAILABLE",
"fadeTime": 500
}
Floorplate ID: Usually Level-{floor} (e.g., Level-9)
POST /api/v1/apartments/all/off
Turn all lights off instantly
POST /api/v1/apartments/all/on
Turn all lights on (optional: {"intensity": 200})

For 500/502 errors, retry with doubling delays: 1s ‚Üí 2s ‚Üí 4s (max 3 retries)
